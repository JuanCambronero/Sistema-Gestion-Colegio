# Sistema Gestión Colegio - Práctica POO Python

Aplicación en consola para gestionar **alumnos**, **profesores**, **asignaturas** y **clases** con menús interactivos, herencia POO, colecciones y excepciones. Incluye memoria PDF con diagrama UML y casos de prueba.
